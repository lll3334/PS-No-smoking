二、赏析练习

元素

正在抽烟的人 → "吸烟"
底下的剪影 → "吸烟的后续"
被烟头烧出的肺部焦痕 → 代表 "肺部损伤、致命危害"

色彩

黑白灰棕、焦褐色
干净的白纸、黑色剪影，唯一的色彩是烧焦的肺部

巧思

一开始看到的是一个抽烟的人，往下看，他的下半部分剪影肺部已经被烧穿。

元素

左半侧：燃烧、冒烟
右半侧：森林与河流

色彩

左侧：红、橙、黑、灰，火焰与浓烟的色调
右侧：绿、蓝、白，森林与河流的色调
整体底色：干净的浅绿色，让对比更突出

巧思

第一眼看到的是左右拼接的肺部，通过 "一毁一荣" 的强烈对比，具象成了看得见的 "毁灭 vs 新生"。

三、 字体设计练习

“拒” → 拒绝 → 严肃、警告

【严肃 → 态度强硬 → 巨石/金属】 【警告 → 暗示危险 → 骷髅元素】

“绿意” → 自然生机 → 治愈、希望

【治愈 → 柔和温暖 → 阳光】 【希望 → 生机 → 嫩芽】

四、 参考效果库

这里为你提供了三种最常用于禁烟海报的特效。挑选适合你创意的特效，参考下列步骤进行合成与尝试。

一：火焰效果

表现烟雾缠绕主体、吞噬健康的立体感

1. 点调整图层
2.点击渐变映射
2. 调整火焰颜色
3. 点击涂抹工具
涂抹出火焰效果

二：烟雾弥散

隐喻生命或钞票随烟草一同燃烧

1. 点击滤镜→模糊画廊
路径模糊
2. 调整曲线
调整参数达到烟雾效果

三：灰烬飞散粒子

体现抽烟等于时间与健康的灰飞烟灭

1. 在上一张的烟雾基础上
添加溶解效果
2. 可叠加之前的图
图层蒙版做擦除效果
